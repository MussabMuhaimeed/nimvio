# Nimvio

**Your curious desktop companions.**

رفاق فضوليون لسطح مكتب ويندوز، يتحركون بين جميع الشاشات المتصلة ويزورون الحواف والزوايا عشوائياً.

## السلوك الذكي

- تبحث الشخصية بعينيها عن مكان مناسب ثم تمشي أو تقفز إليه.
- تفضل أحياناً الحافة السفلية أو إحدى الزوايا كي تجلس هناك.
- بعد الجلوس قد تشير بيدها، تلوّح، تفكر، تنظر حولها، أو تنام.
- عيناها تتابعان مؤشر الفأرة، ثم تنهض لاحقاً وتختار مكاناً أو شاشة أخرى.
- كل دورة مختلفة ومدتها عشوائية كي لا تبدو الحركة مكررة.

## ما الجديد في النسخة الذكية

- عقل داخلي يحسب الطاقة والفضول والملل والسعادة ويختار السلوك بناءً عليها.
- تمييز النوافذ المفتوحة والنافذة النشطة، والجلوس على حواف النوافذ أو سطح المكتب.
- ذاكرة لآخر ستة أماكن وشاشة مفضلة، مع تجنب تكرار المكان مباشرة.
- تتبع مؤشر الفأرة، التفاعل عند النقر، والاستجابة للرمي والسحب.
- حركات انتقالية للجلوس، رمش، تنفس، ميلان الرأس، وظل متغير أثناء القفز.
- أحداث نادرة: التعثر، الدهشة، مطاردة المؤشر، وإخراج منظار للتفقد.
- إعدادات للحجم والنشاط والاستقلالية والشاشات المسموحة ووقت الهدوء وملء الشاشة.
- دعم ثلاث شخصيات بأسماء وألوان وطباع مختلفة، وتتفاعل الشخصيات عند التقارب.
- أيقونة بجانب الساعة: نقرتان لاستدعاء جميع الشخصيات إلى مؤشر الفأرة.

## التثبيت

1. فك ضغط الحزمة.
2. انقر مرتين على `Install.cmd`.
3. سيُنسخ التطبيق إلى `%LOCALAPPDATA%\Nimvio` ويُنشأ اختصار على سطح المكتب.

يتطلب Windows 10/11 و.NET 10 Desktop Runtime. إذا كان .NET SDK 10 مثبتاً فلديك المتطلب مسبقاً.

## التحكم

- اسحب الشخصية بزر الفأرة الأيسر.
- نقرتان لإيقاف الحركة أو متابعتها.
- الزر الأيمن لفتح السرعة، اختيار شاشة، التشغيل مع ويندوز، أو الخروج.

## البناء

```powershell
dotnet publish .\Nimvio.csproj -c Release --no-self-contained
```
